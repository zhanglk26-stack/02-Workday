# 2026 工作日看板

针对 2026 年的中国工作日历工具，含法定节假日、调休安排、备忘录和待办事项。部署于 Cloudflare Pages，零构建、零后端依赖。

## 功能

- **日历视图** — 月历展示，标注工作日 / 休息日 / 节假日 / 调休
- **工作进度** — 当月 & 全年已工作天数、进度条、假期余额
- **节假日倒计时** — 展示距下一个假期还剩多少个工作日
- **备忘录** — 三栏（昨天 / 今天 / 明天）日记，800ms 防抖自动保存
- **待办事项** — 轻量 Todo，持久化到 localStorage
- **亮 / 暗主题** — 一键切换，记忆用户偏好

## 节假日数据

节假日和调休日期来源于国务院每年发布的官方通知，手动更新至 `workday.js`。

> 每年政府发布新通知后，只需更新 `workday.js` 中的 `DATA.h`（节假日）和 `DATA.w`（调休）两个对象即可。

## 文件结构

```
02-Workday/
├── index.html      # HTML 结构
├── style.css       # 全部样式（含亮/暗主题、响应式）
├── app.js          # 应用逻辑
├── workday.js      # 工作日判断核心 + 2026 年假期数据
└── tests/
    └── workday.test.js   # 单元测试（Node.js 内置 test runner）
```

## 本地运行

直接用浏览器打开 `index.html`，或使用任意静态服务器：

```bash
npx serve .
```

## 测试

```bash
node --test tests/workday.test.js
```

## 部署

推送到 Git 仓库后，Cloudflare Pages 自动部署。无需构建命令，发布目录设为根目录 `/` 即可。
